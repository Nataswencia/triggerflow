## Правила для всех GPT-промптов (обязательно)
- Обращение на "вы"
- НЕ использовать длинное тире (-), только дефис (-)
- Делай абзацы и отступы по смыслу - текст должен дышать
- Каждая новая мысль - новый абзац с пустой строкой
- Используй эмодзи по смыслу (не в каждый абзац)
- НЕ начинай с "В мире...", "В современном мире...", "В эпоху..."
- НЕ используй: "Давайте разберемся", "Важно отметить", "Стоит подчеркнуть"
- НЕ заканчивай обобщением "Таким образом..."
- Пиши короткими предложениями, как в переписке
- Начинай с факта, цифры или провокации
- Используй разговорный тон: "вот", "кстати", "смотрите", "знаете что"

## Тон автопостинга
- Экспертный но доступный
- Говорим о результатах клиента, не о технологиях
- Каждый пост связан с услугой на сайте

**Why:** GPT по умолчанию пишет канцелярским стилем, клиенты TPE/PME не читают корпоративные тексты.
**How to apply:** Вставлять эти правила в каждый GPT-промпт в n8n воркфлоу автопостинга.
