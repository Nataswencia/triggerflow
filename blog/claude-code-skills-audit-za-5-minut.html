Что такое Claude Code Skills

Смотрите, Claude Code - это AI-агент, который работает прямо в терминале и имеет доступ к файловой системе

Ключевая фича: он читает файл CLAUDE.md и всё из папки rules/ при каждом запуске. Автоматически. Без напоминания

Skill - это набор markdown-файлов с правилами для конкретной задачи. Один раз пишешь - и больше не объясняешь

Кстати, это не магия и не сложная интеграция. Просто файлы с текстом в нужной папке

Как я создала skill для аудита

После первого аудита я выписала всё, что объясняла Claude вручную. Получился список на 32 пункта

Потом разложила по файлам - каждый файл отвечает за свою зону ответственности

protocol.md
21 критерий с бенчмарками. Что считается хорошим показателем, что - красным флагом. Шкалы оценки по каждому блоку.
pdf-style.md
Грамматика, CSS, верстка. Без точек. Каждое предложение с новой строки. Дефис, не длинное тире. Цвета, отступы, шрифты.
process.md
Пошаговый процесс генерации. Грабли, которые уже наступала. Антипаттерны - что НЕ делать.
delivery.md
Шаблоны сопроводительных сообщений клиентам. Тон, структура, что включать. Поддерживающее, а не отчётное.
verify-accounts.md
Правило: ВСЕГДА проверять что @аккаунты-примеры существуют. Однажды вставила несуществующий аккаунт - неприятно.
examples.md
Аудиты-эталоны из опыта. Примеры хорошей формулировки. Уроки - что клиенты отмечали как особенно полезное.

Почему это работает лучше чем промпт в чате

Вот в чём разница. В обычном чате контекст сбрасывается. Вы написали правила в начале разговора - через 20 сообщений Claude может их "забыть" или интерпретировать иначе

Файлы в rules/ - это другое. Они загружаются каждый раз при старте. Они всегда на месте. Их можно версионировать, дополнять, исправлять

И самое главное - они отделены от конкретного проекта. Хотите применить те же правила к другому аккаунту? Просто говорите "аудит @другой_аккаунт". Система уже знает, что делать

FAQ

Что такое Claude Code Skills?

Claude Code Skills - это набор правил, шаблонов и инструкций в файлах rules/, которые Claude читает автоматически при каждом запуске. Вместо того чтобы каждый раз объяснять "без точек в конце", "проверь @аккаунты", "21 критерий" - всё это лежит в файлах и применяется само

Сколько стоит такой аудит Instagram?

Первый аудит - бесплатно. Напишите ИНСТА в комментариях под любым постом в Instagram - пришлю детали в директ. Полный разбор по 21 критерию в течение 24 часов

Можно ли сделать skill для своего бизнеса?
